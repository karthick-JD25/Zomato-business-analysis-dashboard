## Project Overview
This project analyzes Zomato restaurant data to identify key business insights related to customer preferences, pricing patterns, ratings, and location-based performance. The objective is to support strategic decision-making for restaurant expansion and service improvement.

## Problem Statement
The food delivery industry is highly competitive. Businesses need data-driven insights to understand customer behavior, popular cuisines, pricing trends, and rating patterns.

## Tools Used
- Excel
- SQL


## Key Business Insights
- Identified top-performing restaurant locations
- Analyzed relationship between ratings and price range
- Identified most popular cuisines
- Evaluated customer preferences based on city trends
- Identified factors influencing restaurant success

## Features
- Interactive dashboard with filters
- KPI metrics for performance evaluation
- Location-based analysis
- Rating distribution visualization
- Price category insights

The insights derived from this analysis help stakeholders:

• Identify high-demand cuisines and customer preferences  
• Optimize pricing strategies based on rating trends  
• Select profitable locations for restaurant expansion
